# Labels in assembly code

   | Label | Description |
   | ---| ---|
   | LC0   | local constant,ex) LC0, LC1,... |
   | LFB0  | local function beginning |
   | LFE0  | local function ending |
   | LBB0  | local block beginning |
   | LBE0  | local block ending |
   | L prefix| local labels |
